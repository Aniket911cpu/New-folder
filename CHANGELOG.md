# Changelog

All notable changes to the "FullPageCapture" extension will be documented in this file.

## [1.1.0] - Unreleased
### Added
- **PDF Export**: Added ability to save captures as PDF (via browser print).
- **Keyboard Shortcuts**: Added `Alt+Shift+P` to trigger capture instantly.
- **Modern UI**: Completely redesigned the Popup and Result pages with a modern, clean aesthetic.
- **History View**: Added a list of recent captures.

### Changed
- Improved sticky header hiding logic.
- Optimized image stitching performance.
- Enhanced 32k pixel limit handling.

## [1.0.0] - 2025-12-27
### Added
- Core full-page capture functionality.
- Automatic scrolling and stitching.
- Handling for sticky headers (hide/restore).
- Support for high-DPI displays.
- Automatic splitting of very tall images (>32k px).
- PNG download support.

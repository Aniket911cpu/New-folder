# SnapFlow - Web Capture & Snip

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

**SnapFlow** is a modern, powerful Chrome Extension for capturing web pages. It goes beyond simple screenshots, offering full-page scrolling capture, region selection, and visible area snipping, all wrapped in a beautiful, premium interface.
